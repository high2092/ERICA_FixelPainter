## What's this?
픽셀아트 그리기에 최적화된 그림판

## How to use

1. ERICA_PixelPainter/Third/에 있는 파일을 로컬 저장소의 한 폴더 안에 내려 받습니다.
2. 터미널에서 해당 로컬 저장소의 경로에 접근한 후, 다음 명령어를 입력합니다. (jdk 및 java가 설치되어 있어야 합니다.)

        javac Main.java 
        java Main
